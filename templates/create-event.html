💡 Ingestion Guidelines

Idempotence Key: The Event ID serves as a unique signature. If the network drops and the request is retried, the engine will block processing duplicates, returning a conflict status.

Purchase Calculation: Base rate awards 10 points per 1.00 USD of the transaction amount. E.g. a $15.50 Latte purchase grants 155 points.

Maximum Cap: Maximum point limit per single purchase event is capped at 500 points to prevent abuse.
